# 🚀 TaskFlow

TaskFlow, kişisel ve ekip bazlı görevleri takip edebileceğiniz tam teşekküllü (Full-Stack) bir görev yönetim sistemidir. 

## 🛠️ Kullanılan Teknolojiler
- **Frontend:** React 18, Next.js, Tailwind CSS, Lucide React
- **Backend:** Node.js, Express.js

## 📦 Proje Yapısı
- `/taskflow-backend`: API katmanını içeren Node.js sunucusu. (Port 3001)
- `/taskflow-dashboard`: Kullanıcı arayüzünü barındıran Next.js frontend'i. (Port 3002)

## ⚙️ Hızlı Başlangıç

### Backend
```bash
cd taskflow-backend
npm install
node index.js
```

### Frontend
```bash
cd taskflow-dashboard
npm install
npm run dev -- --port 3002
```
